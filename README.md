# 🏆 Lamu Sports Hub

A modern football management platform for Lamu, Kenya.

## Features
- Live match tracking
- Team management (Lamu Queens, 11 Brothers, Blue Rangers)
- Pitch booking system (Kibaki Grounds, Mkunumbi Stadium)
- Talent scouting network
